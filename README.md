# Pycross
## Pour lancer le jeu:
1) installez python et installez le module:
  - pillow
2) Lancez Pycross.py

## Pour créer son propre picross:
- créer un dossier intitulé Mes_images
- créer le dessin qu'on souhaite
- appuyez sur importer une image et prendre l'image dans le dossier que vous avez créer
